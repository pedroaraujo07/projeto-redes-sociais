# Projeto Redes Sociais

## Sobre o Projeto

O Projeto Redes Sociais é uma aplicação desenvolvida com HTML5 e CSS3 que simula a navegação entre diferentes redes sociais por meio da tela de um smartphone.

A interface apresenta um iPhone centralizado na página e, ao lado, botões com ícones das redes sociais. Ao clicar em um desses botões, o conteúdo correspondente é carregado dentro da tela do aparelho utilizando **iframes**, proporcionando uma experiência interativa e organizada.

O objetivo deste projeto foi colocar em prática conceitos importantes de desenvolvimento front-end, explorando recursos de HTML e CSS para criar uma interface moderna e funcional.

---

## Funcionalidades

- Interface inspirada em um smartphone.
- Navegação entre diferentes redes sociais.
- Carregamento de páginas utilizando iframes.
- Botões laterais com efeitos visuais.
- Layout organizado e responsivo.
- Estrutura sem utilização de JavaScript.

---

## Tecnologias Utilizadas

- HTML5
- CSS3

---

## Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados diversos conceitos, como:

- Estruturação semântica com HTML5;
- Posicionamento de elementos utilizando CSS;
- Uso de `iframe` para exibição dinâmica de páginas;
- Pseudo-classes (`:hover`);
- Transições e efeitos visuais;
- Backgrounds personalizados;
- Organização de arquivos do projeto;
- Responsividade básica;
- Boas práticas de estilização.

---

## Objetivo

Este projeto foi desenvolvido com fins de estudo, buscando praticar técnicas de construção de interfaces modernas utilizando apenas HTML e CSS, além de explorar o funcionamento dos iframes na navegação entre páginas.
